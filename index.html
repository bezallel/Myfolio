ISOLDE – Intelligent Simulation of Load Distribution in Emergencies

  • Category: Energy resilience / grid simulation
  • Technical Skills: Python (Flask), data modeling, real-time UI, Chart.js

Motivation: ISOLDE was built in response to the severe outages caused by Storm Éowyn in Ireland, where large numbers of customers lost power and ESB Networks reported extensive grid damage. The event highlighted the need to understand how stored energy could support communities when infrastructure fails.

Overview: ISOLDE is a real-time simulator that models how a micro-grid or community battery system responds to a major outage. It shows daily load behavior, how storage accumulates during low demand, and how stored energy is allocated to critical areas when the grid goes down.

How It Works: A Flask backend provides load profiles and station metadata through "Data" and "Stations". The frontend displays a live Chart.js load graph, a battery state indicator, and a panel showing which stations receive stored energy during Island Mode.

Impact: The simulator helps utilities, planners, and communities test emergency energy strategies and evaluate priorities, required storage capacity, and how long support can realistically last.

Disclaimer: This is a demonstration project that uses synthetic or publicly available data and is not an operational system.
